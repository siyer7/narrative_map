# narrative_memory

* **Empirical**. Imagine a continuous stream of info that can be segmented at many locations based on [prediction errors](). However, where these segmentation boundaries appear, and concurrently, which of the possible segments/events are linked together neurally ([reinstatement]()) and behaviorally ([recall]()), is also a function of narrative coherence. This can be studied using different stories altogether, which have inherently varying narrative arcs, or using a single story with [multiple narratives](). We use the latter approach as it offers more control.
* **Model components**. Segmenting (PE-based), current event (local memory), narrative (global memory), comparison (predicting event|narrative), linking (narrative update?)
* **Predictions**. Replicative: boundaries, updated narrative/recall; testable: forgetting
* **Notes**. You can initialize each component as an LLM, and gradually make them more computationally honest
